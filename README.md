      CHATBOT (HIMATIF AI)
ide: merancang dan membangun chatbot (HIMATIF AI) untuk membantu maba IT yang berguna untuk menanyakan seputar himatif atau teknologi informasi seperti struktur himatif terbaru, manfaat masuk himatif, dosen matkul TI, sejarah himatif, jadwal event, dan lain-lain.
tujuan: membuat asisten digital himatif yang dapat menjawab pertanyaan mahasiswa teknologi informasi berdasarkan data resmi kampus
target utama: maba IT untara
fitur: 1.UI/UX sederhana 2.Tanya jawab 3.FAQ 4.Informasi kegiatan 5.Jadwal 6.Kepengurusan 7.dll
target lain : kalangan umum (yang tertarik pada himatif dan teknologi informasi untara)
biaya : belum diketahui
AI ini memiliki scoupe(batasan untuk menjawab hanya seputar data yang dimiliki, jika diluar dari data yang ada maka chatbot dapat menjawab "maaf data saat ini belum ditemukan, silahkan tanyakan langsung pada pengurus himatif," atau "maaf kami hanya melayani informasi terkait himatif" atau lainnya). Data yang diberikan ke AI disarankan dalam bentuk pdf, word, excel, atau sejenisnya, supaya AI dapat digunakan untuk jangka panjang, lebih profesional, efisiensi waktu, dan mudah untuk memodifikasi data tanpa harus mengubah algoritma AI.

workflow:
A(user bertanya) → B(apakah sapaan?) 
B → |Ya| C(langsung jawab)
B → |Tidak| D(apakah pertanyaan umum?)
D → |Ya| E(program jawab cepat)
D → |Tidak| F(program mencari data)
F → G(data ditemukan?)
G → |Ya| H(kirim data ke AI)
G → |Tidak| I(beri tahu data tidak ditemukan)
J → K(AI menyusun jawaban)
K → L(user)
